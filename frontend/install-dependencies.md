# Install Dependencies

To fix the lint errors, you need to install the dependencies. Run these commands in the frontend folder:

```bash
npm install
```

Or if you prefer yarn:
```bash
yarn install
```

This will install:
- axios (for API calls)
- clsx (for className utility)
- tailwind-merge (for merging Tailwind classes)
- tailwindcss (for CSS framework)
- @tailwindcss/typography (for typography plugin)
- autoprefixer (for CSS prefixes)
- postcss (for CSS processing)
- @headlessui/react (for UI components)
- @heroicons/react (for icons)

After installing dependencies, the TypeScript errors and Tailwind warnings will be resolved.
